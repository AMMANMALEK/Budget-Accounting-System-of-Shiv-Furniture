# Budget Accounting System of Shiv Furniture

This project is a budget and accounting management system for Shiv Furniture.
The frontend is built using **React + Vite**.

## Tech Stack
- React
- Vite
- JavaScript
- CSS

## Development Setup

This project uses the standard React + Vite setup with Hot Module Replacement (HMR)
and ESLint for code quality.

### Vite Plugins
- `@vitejs/plugin-react` – uses Babel for Fast Refresh
- `@vitejs/plugin-react-swc` – uses SWC for Fast Refresh
